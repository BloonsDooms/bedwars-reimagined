## Core Design
Instead of mid generators, there are mid control points. Holding these give various buffs, such as upgrading your generator.

All items are rebalanced, to provide more linear upgrades.

Frequent consumables such as wool or arrows, are replaced with regenerating items (which can be upgraded, increasing regen speed and stack size)

Gameplay is streamlined as much as possible (a singular tool, for example) while maintaining creative expression.
